上記の設定に加え、今回はメール認証＆Login機能をつけたいので
開発用に `Mailhog` コンテナを使用する用に `docker-compose.yml` に追記、


```

  mailhog:
    image          : mailhog/mailhog:latest
    container_name : mailhog
    hostname       : mailhog
    ports:
      - "1025:1025"
      - "8025:8025"
```
